# API de Gerenciamento (Usuários, Produtos e Categorias)

API REST para gerenciar usuários, autenticação, categorias e produtos (CRUD), desenvolvida como projeto prático individual do semestre para consolidar conhecimentos em desenvolvimento backend.

## Tecnologias

* Java 21
* Spring Boot
* Spring MVC
* Spring Data JPA
* Spring Boot Validation
* MySQL (via XAMPP)
* Maven

## Práticas adotadas

* API RESTful
* Padrão DTO (Data Transfer Object) para proteção e isolamento de dados
* Injeção de Dependências via Construtor
* Validações de regras de negócio
* Tratamento de exceções e respostas padronizadas de Status HTTP (200, 201, 400, 401, 404)
* Consultas e persistência com Spring Data JPA
* Código limpo e separação de responsabilidades (Controllers, Services e Entities)
